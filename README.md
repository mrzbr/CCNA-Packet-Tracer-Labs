Every CCNA home labs that I have created!

## Lab 1: Connecting Devices
* **Objective:** Learned how to pick the correct cables (straight-through vs crossover) and verify interface statuses.
* **Key Commands Used:** `show interfaces`, `show ip interface brief`
* **Notes:** Had to remember that routers and switches use different default interface states out of the box!
